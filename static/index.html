About Me

The Story Behind The Data

I'm Ayoade Adegbite — an Analytics Engineer, international speaker, and the founder of Behind The Data.

My career started in Mechatronics Engineering in Nigeria, but I quickly discovered that the most impactful problems weren't hardware problems — they were data problems. Since then, I've worked across Nigeria, Canada, the UK, the US, and Poland, helping organizations turn messy, undocumented data systems into production-ready analytics platforms.

"I don't just build systems — I build people. Through academies, bootcamps, and mentorship programs across three continents, I've trained hundreds of analysts and engineers."

At The Bredge, I build custom dashboards tracking dbt model usage and Snowflake costs, and I've automated documentation using LLMs — increasing coverage by 35%+. Previously at YipYip Media, I migrated legacy stored procedures to modular dbt models, reducing maintenance time by 45%.

Currently pursuing my Master's in Information Technology at Miva Open University, and I hold certifications in Tableau and Apache Airflow (Astronomer DAG Authoring).

Open Source

Tools I've Built

Open-source dbt packages solving real problems in analytics engineering — documentation, data quality, and observability.

dbt-doc-tracker

Detects and records documentation changes across your dbt project. Records only the diffs — when a description is added, modified, or removed. Built for documentation governance and compliance tracking.

★ 6 SQL / Jinja
dbt-doc-inherit

Propagates column descriptions along the DAG — define once, inherit downstream. Eliminates repetitive documentation of the same column across dozens of models.

★ 3 SQL / Jinja
dbt-autodoc

Auto-generate dbt model and column descriptions using heuristics or LLMs. The tool behind the 35%+ documentation coverage improvement at The Bredge.

Python
dbt-anomaly-detector

Lightweight dbt package for statistical anomaly detection on warehouse metrics using Python models. Catch data quality issues before they hit dashboards.

★ 1 Python / SQL
tap-substack

Singer tap for extracting Substack newsletter data. Built to support analytics on newsletter performance and subscriber engagement.

★ 1 Python
awesome-modern-data-stack

Curated list of open-source tools covering every layer of the modern data stack — ingestion, warehousing, transformation, orchestration, quality, cataloging, and more.
